PENMSH - A Fortran Tool for Creating Hexahedral Meshes in PENTRAN and TITAN Codes

Download this project as a .zip file Download this project as a tar.gz file

Introduction

PENMSHXP is a 3d geometry builder for PENTRAN, TITAN, and MCNP(5/6).

Installation

Installation Notes Here

To install PENMSHXP (Makefile tested solely with Linux). Currently gfortran or portland Fortran compilers are supported with the Makefile. Ideally this code should be installed on a parallel HPC where PENTRAN and TITAN codes are run. 

$ git clone git@github.com:critcel 
$ make TARGET=gfortran 
$ ./penmshxp -h # to check compilation and see features
